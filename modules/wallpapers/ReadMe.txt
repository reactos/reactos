To include the module in your build folder, run the configure script with the flags -DENABLE_WALLPAPERS=1

# For Windows users

    configure.cmd -DENABLE_WALLPAPERS=1

# For UNIX users

    ./configure.sh -DENABLE_WALLPAPERS=1
