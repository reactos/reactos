Remote Access

When you use remote access, Windows 2000 can enable users to dial in from any location and gain access to your internal network. Use the Routing and Remote Access Configuration wizard to set up this service.
 
To start the Routing and Remote Access Configuration Wizard
1. Open Routing and Remote Access.
2. In the console tree, expand Routing and Remote Access, right-click your server, and click Configure and Enable Routing and Remote Access.
