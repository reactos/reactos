Component Services

Component Services provides comprehensive services for component applications. These services are already installed with Windows 2000 Server, and include:

  • Resource Pooling — Maximizes efficiency in the the use of system and database connection resources.
  • Security — Integrates and manages security for your applications.
  • Transaction Services — Guarantees distributed data consistency.
  • Event Services — Publishes and subscribes eventing between components.
  • Queued Components — Queues messages for component applications.
  • In-Memory Database — Accesses frequently used data with memory speed.
  • Application Management — Deploys, configures, and monitors your component applications.
