Terminal Services

Start using Terminal Services by choosing from the following administration tools:
 
Terminal Services Configuration —  Creates new connections and enables you to modify the settings of existing connections.

Add/Remove Programs —  Ensure applications are installed for multisession use.

Terminal Services Client Creator — Makes installation disk sets for client terminals.

Terminal Services Manager —  Displays all of the terminal servers and domains running on your network.

Terminal Services Licensing —  Installs the licensing infrastructure to ensure license compliance.
