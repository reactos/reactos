Message Queuing

Message Queuing is a communication infrastructure that enables distributed messaging applications running at different times to communicate across platforms and networks that may be temporarily offline.

Note
Active Directory must be available on your network to complete Message Queuing Setup unless you plan to join a Windows NT 4.0 MSMQ enterprise or a workgroup.

  • If you are part of a Windows 2000 domain —  Message Queuing must be installed on at least one domain controller in the current site before you can install Message Queuing on other servers.
  • If you plan to join a Windows NT 4.0 MSMQ enterprise —  You must provide the name of an MSMQ primary site controller (PSC) during Message Queuing Setup.
