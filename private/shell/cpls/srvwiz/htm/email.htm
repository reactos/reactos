E-mail Server

Windows 2000 Server provides a reliable, scalable, and stable operating environment for e-mail servers and gateways across multiple protocols and client platforms. Using the latest technology, Windows 2000 can optimize your client connection speeds and reduce network congestion while maintaining high levels of security.

Windows 2000 does not need further configuration. However, you need to install an e-mail server to take advantage of Windows 2000 messaging capabilities.
